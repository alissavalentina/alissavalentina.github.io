One Page Design

CONCEPT

A client that owns a restaurant has come to you looking for your website company to design a ONE PAGE layout of their current menu page.
Their customers have been having a hard time remaining on the page while searching. They have provided you with the basic outline which includes the HTML of the site. They are looking for you to design it to be more appealing to the customers.

THE ACTIVITY

  • Brainstorm how YOU want the layout to be!
  • This does NOT have to be handed in. It is for your own brainstorming
  • Think of a colour scheme, typography and overall theme you want your website to have.

THE TASK

  • Go to this LINK and download the raw upload.html file
  • Add the file to your presentations/week9 folder in Atom.
  • Using the HTML already provided, style the site.
  • Feel free to make changes to the already styled elements of the page if need be.
  • Using the classes already provided to you, link to the different parts of the page. By doing this you are ensuring that your site is a ONE PAGE DESIGN

REQUIREMENTS

  1. Anchor to the different parts of the page.
  2. Have a cohesive theme.
  3. Place your name in the title
  4. Keep the content of the HTML the same.
  5. Include at least two images.
  6. Be creative & have fun!

Additions if wanted!

  • If you'd like you can add JavaScript to the page to give it an additional piece of creativity.

UPLOAD

  • When complete. Link the file to yourmain index.html page.
  • When done, commit and push to origin.
