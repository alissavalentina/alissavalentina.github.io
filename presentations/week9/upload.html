One Page Design

CONCEPT

A client that owns a restaurant has come to you looking for your website company to design a ONE PAGE layout of their current website.
They have provided you with the basic outline which includes the HTML of the site. They are looking for you to design it to be more appealing to the customers.

THE ACTIVITY

  • Brainstorm how YOU want the layout to be!
  • This does NOT have to be handed in. It is for your own brainstorming
  • Think of a colour scheme, typography and overall theme you want your website to have.

THE TASK

  • Go to this LINK and download the raw upload.html file
  • Add the file to your presentations/week9 folder in Atom.
  • Using the HTML already provided, style the site.
  • Using the classes already provided to you, LINK to the different parts of the page. By doing this you are ensuring that your site is a ONE PAGE DESIGN

UPLOAD

  • When complete. Link the file to your main index.html page.
  • When done, commit and push to origin.
